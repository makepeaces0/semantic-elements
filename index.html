Elements With Meaning: A Summary of Semantic Elements

Semantic elements are terms used to make the grouping of pages more clearly and universally understood by both the developer and browser. Semantics provide the meaning behind the tags being used instead of having terms that are exclusively understood by the developer. This leads to accessibility, searchability, internationalization, and interoperability according to Michelle in An Overview of HTML5 Semantics (2016). Tags that represent the look of the site are not included in semantics because they arenâ€™t attributed to meaning.

Use semantic tags to enhance:

  • Accessibility to technologies
  • Searchability by computers
  • Internationalization so the world can read it
  • Interoperability to create readable structure
